Overview

Understanding spatial relations is essential for Large Vision-Language Models (LVLMs) to interact with the physical world. Yet real-world visual spatial reasoning (VSR) remains underexplored: strong models still fail on basic geometric queries in complex imagery.

We introduce ReSynth (Real-World Synthetic Spatial Reasoning Dataset): a benchmark with real-world evaluation data and scalable synthetic training data. Our studies show that inference-time techniques (e.g., Chain-of-Thought) and model scaling yield only marginal gains on VSR — pointing to a lack of appropriate spatially grounded training data rather than a pure “reasoning deficit.”

To obtain physically consistent, photorealistic supervision, we propose AI-Assisted Rendering (AIR): 3D geometric prerendering for accurate ground truth, plus generative texturing for realism. Training on the resulting synthetic set substantially improves VSR; our fine-tuned SRVLM-8B reaches 68.6% on ReSynth and 71.3% on RealWorldQA (among reported baselines in the paper).

Question types

Real-world images span diverse daily-life scenes; each image has spatially grounded questions with multi-annotator review. The synthetic side pairs template-based questions with answers computed from 3D scene metadata (positions, orientations, bounding boxes).

Question types

1. Relative direction

Target direction relative to a reference in 3D; disambiguate true 3D relations from misleading 2D projections.

2. Absolute direction

Spatial layout from the camera’s fixed perspective using perspective, size, alignment, and context.

3. Link prediction

Relational links between two targets via a reference; includes between-ness and comparative distance.

4. Object size

Size judgments from visual context (perspective, overlap), not prior world knowledge alone.

5. Existence

Detect objects from visual evidence only; avoid spurious contextual assumptions.

6. Counting

Count under occlusion and perspective without scene assumptions.

7. Distance

Separation from visual perspective; avoid scaling shortcuts (e.g., “equally spaced” bias).

8. Common sense

Plausibility from contextual cues (e.g., weather) without dataset-specific bias.

Method

AIR bridges simulation geometry and real-world appearance; SRVLM is built by fine-tuning an LVLM on ReSynth synthetic training data with RL (DAPO) as described in the paper.

Stage 1

Geometric prerendering

Scene presetting, object curation, stochastic material coloring, 3D placement with validation (collision, occlusion), then Blender rendering. Metadata ( scene_pose_dict, color_info) drives downstream QA.

Stage 2

AI texturing

Compositional prompts for indoor/outdoor scenes; image editing (e.g., Qwen-Image-Edit in the paper); automatic filtering with a vision assessor (e.g., Qwen2.5-VL) for quality control.

Stage 3

QA generation

Template-based questions with lexical variation; answers from geometric functions on 3D state (relative direction, distance, counting, size, etc.).

SRVLM-8B is fine-tuned from Qwen3-VL-Thinking-8B on the ReSynth synthetic training set using DAPO (group sampling, decoupled clipping). See the paper for reward design and training hyperparameters.
