Overview

Understanding spatial relations is essential for Large Vision-Language Models (LVLMs) to interact with the physical world. Yet real-world visual spatial reasoning (VSR) remains underexplored: strong models still fail on basic geometric queries in complex imagery.

We introduce ReSynth (Real-World Synthetic Spatial Reasoning Dataset): a benchmark with real-world evaluation data and scalable synthetic training data.

To obtain physically consistent, photorealistic supervision, we propose AI-Assisted Rendering (AIR). Training on this synthetic set substantially improves VSR; our fine-tuned SRVLM-8B reaches 68.6% on ReSynth and 71.3% on RealWorldQA.

Question types

Question Types

1. Relative direction

Target direction relative to a reference in 3D; disambiguate true 3D relations from misleading 2D projections.

2. Absolute direction

Spatial layout from the camera perspective using perspective, size, alignment, and context.

3. Link prediction

Relational links between two targets via a reference, including between-ness and comparative distance.

4. Object size

Size judgments from visual context (perspective, overlap), not prior world knowledge alone.

5. Existence

Detect objects from visual evidence only and avoid spurious contextual assumptions.

6. Counting

Count under occlusion and perspective without scene-level assumptions.

7. Distance

Estimate separation from perspective cues without scaling shortcuts.

8. Common sense

Infer plausibility from contextual visual cues while minimizing dataset-specific bias.
